What we did with our website?

  • This is a website which simulates a rudimentary police dashboard

  • We used the randomusers.me API to generate random people (so, no information given are from real people, other than pictures provided by the API) to fill out a record which can be viewed by clicking the "Show All Records" button.

  • We use a set seed to generate these people, meaning you'll see the same group of people each time.

  • You are able to see more details by searching the person's full name inside the "Search Records" button (as long as you type it correctly in the following format: ["FirstName Surname", "Surname, First Name", "Surname First Name"]).
